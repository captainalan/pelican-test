Haskell

Function application with the <*> operator and a Maybe context:

Just (*2) <*> Just 10
Just 20

Why might you want to do that?

Just (*2) <*> Just "asdf"
-- ERROR....!!!

Things blow up if incompatible types are used. Whoa!

Here's some more functional fun.

(*) <$> [1,2] <*> [3]
[3,6]

The code above maps the multiplication operator to the first list [1,2] to produce the functions something like this [(\x -> x * 1), (\x -> x * 2)]. These functions are in turn each applied to to [3] giving [3,6].
